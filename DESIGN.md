# Design System Document

This document outlines the core visual identity of our product.

## Branding

### Color Palette
- **Color Mode**: `light`
- **Primary Color**: `#10A3FF` - A vibrant blue, used for primary calls to action, interactive elements, and key brand accents.
- **Secondary Color**: `#00E676` - A bright green, providing a complementary accent for secondary actions or informational highlights.
- **Tertiary Color**: `#FF9100` - A bold orange, used for additional emphasis, alerts, or decorative elements.
- **Neutral Color**: `#F0F4F8` - A light grey, serving as the base for backgrounds, surfaces, and less prominent UI components.

### Typography
- **Headline Font**: `plusJakartaSans` - Used for all headings and prominent text elements, ensuring clear hierarchy.
- **Body Font**: `plusJakartaSans` - Applied to all paragraph text, lists, and general content for readability.
- **Label Font**: `plusJakartaSans` - Employed for form labels, small text, and UI annotations.

## UI Elements

### Roundedness
- **Roundedness**: `3` (Maximum, pill-shaped) - All UI elements feature highly rounded, almost pill-shaped corners, creating a soft and friendly aesthetic.

### Spacing
- **Spacing**: `2` (Normal) - A balanced approach to whitespace, providing comfortable readability and visual separation between elements without feeling overly sparse or dense.
